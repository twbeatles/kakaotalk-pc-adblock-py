# KakaoTalk AdBlocker Pro v5.0

카카오톡 PC 버전의 배너 광고를 차단하고, 광고가 있던 자리에 남는 빈 공간(레이아웃)을 자동으로 제거해주는 프로그램입니다.

## ✨ 주요 기능 (v5.0 업데이트)

- **강력한 광고 차단**: Hosts 파일을 기반으로 카카오톡 광고 서버와의 통신을 원천 차단합니다.
- **빈 공간 자동 제거**: 광고가 차단된 자리에 남는 회색 빈 공간을 자동으로 감지하여 채팅 목록을 꽉 차게 늘려줍니다.
- **✨ 스마트 최적화 (New)**: 버튼 한 번으로 `광고 차단` + `DNS 초기화` + `카카오톡 재시작`을 한 번에 수행합니다.
- **모던 UI (New)**: 카카오톡 테마와 어울리는 깔끔한 플랫 디자인이 적용되었습니다.
- **시스템 트레이 지원**: 프로그램을 닫아도 백그라운드에서 조용히 작동하며 시스템 리소스를 거의 사용하지 않습니다.

## 🚀 사용 방법

1. **관리자 권한으로 실행**:
   - 프로그램(또는 exe 파일)을 우클릭하고 **'관리자 권한으로 실행'**을 클릭합니다.
   - (Hosts 파일을 수정하기 위해 필수적입니다)

2. **스마트 최적화 클릭**:
   - 메인 화면의 노란색 **[✨ 스마트 최적화]** 버튼을 클릭합니다.
   - 프로그램이 자동으로 광고 도메인을 차단하고, DNS를 정리한 뒤 카카오톡을 재시작합니다.

3. **확인**:
   - 카카오톡이 재시작되면 친구 목록 하단의 배너 광고가 사라졌는지 확인합니다.
   - 빈 공간 없이 목록이 끝까지 내려오는지 확인합니다.

## ⚙️ 설정 옵션

- **닫을 때 트레이로 최소화**: [X] 버튼을 눌러도 프로그램이 종료되지 않고 트레이 아이콘으로 숨겨집니다.
- **실시간 프로세스 모니터링**: 카카오톡 실행 여부를 감지하여 상태를 표시합니다.
- **광고 레이아웃 자동 제거**: 이 기능이 켜져 있어야 빈 공간(회색 영역)이 자동으로 사라집니다.

## �️ 요구 사항

- **OS**: Windows 10 / 11 (64bit 권장)
- **Python**: 3.8 이상 (소스코드 실행 시)
- **관리자 권한**: 필수

## ⚠️ 주의사항

- 이 프로그램은 카카오톡의 실행 파일 자체를 변조하지 않습니다 (Safe).
- 카카오톡 업데이트에 따라 광고 서버 주소가 변경될 경우, `blocked_domains.txt` 파일을 수동으로 업데이트하거나 새 버전을 다운로드해야 할 수 있습니다.
