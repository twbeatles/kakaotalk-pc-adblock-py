# 카카오톡 광고 차단기 Pro v4.0

카카오톡 PC 버전의 광고를 hosts 파일 기반으로 차단하는 프로그램입니다.

## 📋 주요 기능

- 🛡️ **광고 차단**: hosts 파일을 수정하여 광고 서버 차단
- 🔄 **실시간 모니터링**: 차단 상태 변경 자동 감지
- 💾 **자동 백업**: hosts 파일 수정 전 자동 백업
- 🎨 **시스템 트레이**: 백그라운드 최소화 실행
- 🔔 **Toast 알림**: 비침투적 알림 시스템

## ⌨️ 키보드 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+B` | 광고 차단 시작 |
| `Ctrl+U` | 차단 해제 |
| `Ctrl+R` | 카카오톡 재시작 |
| `F5` | 상태 새로고침 |

## 🚀 사용 방법

1. **관리자 권한으로 실행** (필수)
   - 프로그램 우클릭 → "관리자 권한으로 실행"

2. **광고 차단**
   - "🛡️ 광고 차단 시작" 버튼 클릭

3. **카카오톡 재시작**
   - "🔁 카톡 재시작" 버튼 또는 `Ctrl+R`
